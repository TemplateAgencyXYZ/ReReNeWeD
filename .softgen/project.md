# Recycled Goods E-Commerce Platform

## Vision
A full-featured e-commerce platform for recycled/upcycled goods with dual interfaces: customer shopping experience and admin management dashboard. Built for conscious consumers seeking sustainable products.

## Design
Botanical-renewal aesthetic — clean optimism with natural growth signals.

**Colors:**
- `--primary: 162 67% 18%` (deep forest green #0F4C3A)
- `--accent: 114 16% 60%` (sage green #8BA888)
- `--muted: 114 15% 92%` (pale sage tint)
- `--background: 0 0% 100%` (clean white)
- `--foreground: 180 7% 19%` (deep charcoal #2C3333)

**Typography:**
- Display: Newsreader (elegant serif for products)
- Body: IBM Plex Sans (clean, modern)

**Style:** Clean editorial, generous spacing, natural warmth, optimistic forward-looking rather than rustic nostalgia.

## Features
**User Flow:** Browse products → Add to cart → Checkout with Razorpay → Profile management → Order history → Address management
**Admin Flow:** Product CRUD → User management → Order oversight → Content updates (images, text, pricing, featured status)
**Core:** Supabase auth, database, Razorpay payments
